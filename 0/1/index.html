    1. basic concepts

    ①. 字母表：符号 (字符) 的非空有穷集 $$ \begin{aligned} &\Sigma_1=\{0,1\},\ \Sigma_2=\{a,b,...,z\}\\ &\Sigma_3=\{x\ |\ x\ 是一个汉字\} \end{aligned} $$

    ②. 字符串：由某字母表中符号组成的有穷序列

        若 $\Sigma_1=\{0,1\}$，那么 $0,1,00,111001$ 是 $\Sigma_1$ 上的字符串。

    ③. 空串：记为 $\varepsilon$，有 0 个字符的串。字母表 $\Sigma$ 是任意的，但都有 $\varepsilon \notin \Sigma$。

     符号使用的一般约定：

    • 字母表：$\Sigma, \Gamma,...$
    • 字符：$a,b,c,...$
    • 字符串：$...,w,x,y,z$
    • 集合：$A,B,C,...$

    ④. 字符串 $x$ 和 $y$ 的连接：将首尾相接得到新字符串的运算，记为 $x\cdot y$ 或 $xy$。

       $x\cdot y= \begin{cases} x& y=\varepsilon\\ (x\cdot z)a& y=za \end{cases}$

    ⑤. 字符串的幂

       $x^n= \begin{cases} \varepsilon& n=0\\ x^{n-1}x& n>0 \end{cases}$

    ⑥. 集合 $A$ 和 $B$ 的连接：记为 $AB$ 或 $A\cdot B$

     $A\cdot B=\lbrace w\ | \ w=x\cdot y,\ x\in A \ \mathit{and}\ \ y \in B\rbrace$

    ⑦. 集合的幂

       $A^n= \begin{cases} \lbrace\varepsilon\rbrace& n=0\\ A^{n-1}A& n>0 \end{cases}$

     若 $\Sigma$ 为字母表，那么 $\Sigma^n$ 为 $\Sigma$ 上长度为 $n$ 的字符串的集合。例如，若 $\Sigma=\{0,1\}$，则有
    $\Sigma^0=\{\varepsilon\},\ \Sigma^1=\{0,1\},\ \Sigma^2=\{00,01,10,11\},...$

    $\ \ \emptyset^0=\lbrace\varepsilon\rbrace;\ \ \ \ \forall n \geq 1, \ \ \emptyset^n=\emptyset;\ \ \ \ \ \ \ \ \ \ \emptyset^*=\lbrace\varepsilon\rbrace$
    $\lbrace\varepsilon\rbrace^0=\lbrace\varepsilon\rbrace;\ \ \forall n \geq 1, \ \lbrace\varepsilon\rbrace^n=\lbrace\varepsilon\rbrace;\ \ \lbrace\varepsilon\rbrace^*=\lbrace\varepsilon\rbrace$

    ⑧. 克林闭包 (kleene closure) 与正闭包 $$

    $$ \Sigma^*=\bigcup^\infty_{i=0}\Sigma^i=\Sigma^+\cup\lbrace\varepsilon\rbrace,\ \ \ \ \ \ \Sigma^+=\bigcup^\infty_{i=1}\Sigma^i $$
