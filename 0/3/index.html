    3. NFA

    1. NFA

    在 DFA 的基础上，为其增加非确定性的能力：

    • 给定一个状态和一个字符，可以转移到多个不同的状态
    • 识别串的过程类似于 "fork" 的过程，识别结束后其中一个线程达到接收状态即视为接收

    (1). 定义

    非确定有穷自动机 (Nondeterministic Finite Automaton) $A$ 为五元组 $A=(Q,\Sigma,\delta,q_0,F)$：
     ① $Q$ : a finite set of states；
     ② $\Sigma$ : a finite set of input symbols；
     ③ $\textcolor{blue}{\delta}$ : $Q\times \Sigma\to 2^Q$，transition function；
     ④ $q_0\in Q$ : a start state
     ⑤ $F\subseteq Q$ : a set of final or accepting states。

    例：接收全部以 $01$ 结尾的串

    (2). 扩展转移函数

     $ \hat\delta(q,w)= \begin{cases} \{q\}& w=\varepsilon\\ \bigcup_{p\in\hat{\delta}(q,x)}\delta(p,a)& w=xa \end{cases} $

    (3). NFA 的语言

    $N=\{Q,\Sigma,\delta,q_0,F\}$，则 NFA $N$ 接收的语言为 $\mathbf{L}(N)=\{w\in\Sigma^*\ | \ \hat{\delta}(q_0,w)\cap F\ne \emptyset\}$

    2. $\varepsilon$ - NFA

    在 NFA 的基础上，为其增加空转移的能力，即自发地、不消耗输入字符就发生状态的改变。空转移使得对目标状态的针对性更强，从而简化 NFA 的设计。

    (1). 定义

    带有空转移的非确定有穷自动机 $A$ 为五元组 $A=(Q,\Sigma,\delta,q_0,F)$：
     ① $Q$ : a finite set of states；
     ② $\Sigma$ : a finite set of input symbols；
     ③ $\textcolor{blue}{\delta}$ : $Q\times (\Sigma\cup\lbrace\varepsilon\rbrace)\to 2^Q$，transition function；
     ④ $q_0\in Q$ : a start state
     ⑤ $F\subseteq Q$ : a set of final or accepting states。

    例：设计 $L=\lbrace w\in \lbrace 0,1\rbrace^*\ |\ w$ 倒数三个字符至少有一个是 $1\rbrace$ 的 NFA

    $(1)$ 和 $(2)$ 都在 $q_0$ 消耗掉 $1$ 前的字符，区别在于接受状态的位置。$(3)$ 简化了由 $(1)$ 到 $(2)$ 的过渡。


    (2). 扩展转移函数

    $\small\bullet$ 状态 $q$ 的 $\varepsilon$ 闭包 ($\varepsilon$-$\small\mathit{Closure}$): 记为 $\small\mathrm{ECLOSE}(q)$，表示从 $q$ 经过 $\varepsilon\varepsilon\varepsilon...$ 可达的全部状态的集合，递归表示为：
     ① $\small q\in\mathrm{ECLOSE}(q)$;
     ② $\small\forall p\in\mathrm{ECLOSE}(q)$，若 $\small r\in\delta(p,\varepsilon)$，则 $\small r\in\mathrm{ECLOSE}(q)$。

    $\small\bullet$ 状态集 $\small S$ 的 $\varepsilon$ 闭包： $$\small\mathrm{ECLOSE}(S)=\bigcup_{q\in S}\mathrm{ECLOSE}(q)\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ $$

    $\small\bullet$ 扩展转移函数：

     $\small \hat\delta(q,w)= \begin{cases} \mathrm{ECLOSE}(q)& w=\varepsilon\\ \mathrm{ECLOSE}(\bigcup_{p\in\hat{\delta}(q,x)}\delta(p,a))& w=xa \end{cases} $

    (3). $\varepsilon$ - $\mathrm{NFA}$ 的语言

    $E=\{Q,\Sigma,\delta,q_0,F\}$，则 $\varepsilon$ - NFA $E$ 接收的语言为 $\mathbf{L}(E)=\{w\in\Sigma^*\ | \ \hat{\delta}(q_0,w)\cap F\ne \emptyset\}$

    3. 等价性

    (1). DFA 与 NFA

    定理：语言 $L$ 被 NFA 接受 $\Leftrightarrow$ $L$ 被 DFA 接受

    子集构造法：由 NFA $\small N=( Q_N,\Sigma,\delta_N,q_0,F_N)$ 构造 DFA $\small D=(Q_D,\Sigma,\delta_D,\lbrace q_0\rbrace,F_D)$
     ① $\small Q_D=2^{Q_N}$；
     ② $\small F_D=\{S\ |\ S\subseteq Q_N,\ S\cap F_N\ne \emptyset\}$;
     ③ 对 $\small \forall S\subseteq Q_N,\forall a \in \Sigma,\ $有 $\small \delta_D(S,a)=\bigcup_{p\in S}\delta_N(p,a)$。
    那么有 $\small \mathbf{L}(D)=\mathbf{L}(N)$。

    对于 $\small D$、$\small N$ 以及 $\forall w$，可以通过归纳法证明 $\hat{\delta}_D(\lbrace q_0\rbrace,w)=\hat{\delta}_N(q_0,w)$.
    因此，该算法就是从 $\small N$ 的 $q_0$ 出发，不断选取 $\small S\subseteq Q_N$ 中的元素，扩充 $D$ 的状态的过程。

    (2). DFA 与 $\varepsilon$ - NFA

    与普通 NFA 不同的是，由于自发性的空转移，需要在每一步转移处求状态集的空闭包。

    子集构造法：由 $\varepsilon$ - NFA $\small E=( Q_E,\Sigma,\delta_E,q_0,F_E)$ 构造 DFA $\small D=(Q_D,\Sigma,\delta_D,q_D,F_D)$
     ① $\small Q_D=2^{Q_E}$ 或 $\small Q_D=\lbrace S\subseteq Q_E\ | \ S=\mathrm{ECLOSE}(S)\rbrace$；
     ② $\small q_D=\mathrm{ECLOSE}(q_E)$；
     ③ $\small F_D=\{S\ |\ S\subseteq Q_D,\ S\cap F_N\ne \emptyset\}$;
     ④ 对 $\small \forall S\subseteq Q_D,\forall a \in \Sigma,\ $有 $\small \delta_D(S,a)=\mathrm{ECLOSE}(\bigcup_{p\in S}\delta_E(p,a))$。
    那么有 $\small \mathbf{L}(D)=\mathbf{L}(E)$。

    4. More Examples

    (1). Design NFA:
    ① $L=\lbrace w\in\lbrace 0,1\rbrace^*\ | \ w$ 的首尾字符相同 $\rbrace$；
    ② $L=\lbrace w\in\lbrace 0,1\rbrace^*\ | \ w\ \mathrm{either\ begins\ or\ ends\ with\ 01}\rbrace$；

    (2). Design $\varepsilon$-NFA:
    ① $\lbrace 0^k\ | \ k\ \mathrm{is\ a\ multiple\ of\ 2\ or\ 3}\rbrace$
